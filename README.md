Customer Churn Analysis 📊

📌 Project Overview

This project analyzes customer churn behavior in a telecom company and builds a predictive model using machine learning.

🎯 Objectives
	•	Understand factors influencing customer churn
	•	Perform exploratory data analysis (EDA)
	•	Build a predictive model using Logistic Regression

🛠️ Tools & Technologies
	•	Python (Pandas, NumPy)
	•	Visualization (Seaborn, Matplotlib)
	•	Machine Learning (Scikit-learn)

📊 Key Insights
	•	Customers with month-to-month contracts are more likely to churn
	•	Long-term customers (higher tenure) are less likely to leave
	•	Fiber optic users show higher churn behavior

🤖 Model Performance
	•	Accuracy: ~79%
	•	Recall (Churn): ~52%

📁 Project Structure
	•	data/ → dataset
	•	notebook/ → analysis notebook

🚀 Future Improvements
	•	Improve recall for churn class
	•	Try advanced models (Random Forest, XGBoost)
